# 📚 Wordgaze

Bienvenido a **Wordgaze**: tu compañero perfecto para aprender y mejorar tu inglés de forma interactiva y efectiva. 🌟

---

## 🌟 Características principales

- **Traducción rápida**: Traduce palabras y frases de inglés a español y viceversa.
- **Aprendizaje personalizado**: estudia vocabulario para memorizar palabras nuevas.
- **Pronunciación**: Escucha la pronunciación correcta de cada palabra para perfeccionar tu acento.
- **Diseño intuitivo**: Compatible con dispositivos móviles y escritorio para que puedas aprender en cualquier lugar.

---

## 🛠️ Tecnologías utilizadas

Este proyecto fue desarrollado utilizando:

- **Frontend**: HTML5, CSS3, JavaScript.
- **Backend**: PHP.
- **Base de datos**: MySQL para mostrar listas de vocabulario.
